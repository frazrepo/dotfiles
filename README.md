# Installation
## tmux
* Install [Tmux Plugin Manager](https://github.com/tmux-plugins/tpm)
* Copy .tmux.conf and .tmux.conf.local to home directory
## vim and neovim
* Find instructions here [https://github.com/frazrepo/vimrc](https://github.com/frazrepo/vimrc)
## zsh
