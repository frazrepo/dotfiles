# init.lua or init.vim
Use init.lua. The init.vim configuration is deprecated


# Settings VSCode

For performance optimization, put these lines in vscode settings : 

```json
    "extensions.experimental.affinity": {
        "asvetliakov.vscode-neovim": 1
    },

```
